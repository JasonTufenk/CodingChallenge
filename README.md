# Coding Challenge

## Instructions

### Set Up

Run the following command:

`yarn install`

### Testing

Run the following command:

`yarn mocha`

### Test Cases

| scenario                      | type     |
| ----------------------------- | -------- |
| lowercase palindrome          | positive |
| lowercase non-palindrome      | negative |
| palindrome with uneven casing | positive |
| null                          | negative |
| empty string                  | negative |
| palindrome with punctuation   | positive |
